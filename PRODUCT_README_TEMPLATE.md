# Installation Guide - [Website Name]

Thank you for purchasing this Starter Site from Websites Arena!

## 1. What's Inside
- `/src` - The source code for the website.
- `/public` - Images and assets.
- `package.json` - The dependencies list.

## 2. How to Install
1.  Unzip this folder.
2.  Open your terminal/command prompt in this folder.
3.  Run `npm install` to install dependencies.
4.  Run `npm run dev` to start the site locally.
5.  Open [http://localhost:3000](http://localhost:3000).

## 3. How to Deploy (Go Live)
The easiest way to host this site for free is **Vercel**.
1.  Create a GitHub repository and push this code.
2.  Go to [Vercel.com](https://vercel.com) and import your repository.
3.  Click **Deploy**.
